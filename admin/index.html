چوونەژوورەوەی ئەدمین

تەنها بۆ خاوەنی هەژمار (Supabase Auth)

چوونەژوورەوە کار ناکات؟
  1. Supabase → Authentication → Users → Add user (Auto Confirm چالاک)
  2. Providers → Email → Enable
  3. URL configuration → Site URL: https://toktik.lol/
  4. Redirect URLs: https://toktik.lol/admin/
  5. هەمان ئیمەیڵ/وشەی نهێنی لێرە بنووسە
